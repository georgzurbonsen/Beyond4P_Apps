2. Access Local B4P Documentation and Function Descriptions
_______________________________________________________________________________

list functions                  Lists all procedures and functions
list functions ( '*if*' )       Lists all functions containing 'if'

explain functions               Lists all procedures and functions
explain functions ( 'c*')       Explain all of functions starting with 'c'
explain functions ( for )       Explain the 'for' procedure

dump functions( [table name] )  Dumps all functions with details in a table
